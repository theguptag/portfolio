Must-Have Chrome Extension as a Web Developer

April 26,2022. 3min read

CSS Viewer

The CSSViewer extension is particularly useful for web developers as it displays the parameters of a web page element.



A little popup window emerges when you hover your mouse over an element, displaying the CSS data that makes up the element you're pointing at. Wherever you point your mouse, this addon rapidly identifies crucial CSS properties.



GoFullPage

This extension allows you to take a high-quality screenshot of any entire web page without needing to compile individual screenshots. To snap a high-quality screenshot of a web page, simply install the extension and click the button.



What Font

What font is a simple and effective chrome extension that is very useful for developers who may need to identify fonts used on other web pages, it works in a matter of seconds. It identifies the font family, size, font-weight and color, all within a small browser pop-up window.


So, these are the chrome extensions I strongly recommend you install to make your development trip more efficient. Thanks!
