Must-Have Chrome Extension as a Web Developer

April 26,2022. 3min read

CSS Viewer

The CSSViewer extension is particularly useful for web developers as it displays the parameters of a web page element.



When you hover your mouse over an element, a small popup window appears that displays the CSS data that makes up the element you're pointing at. This extension quickly identifies key CSS properties wherever you point your mouse.



GoFullPage

This extension allows you to take a high-quality screenshot of any full web page without having to take multiple screenshots and combine them. Simply install the extension and click the button to take a high-quality screenshot of a web page.



What Font

What font is a simple and effective chrome extension that is very useful for developers who may need to identify fonts used on other web pages, it works in a matter of seconds. It identifies the font family, size, weight and color, all within a small browser pop-up window.


So, these are the chrome extensions I strongly recommend you install to make your development trip more efficient. Thanks!
